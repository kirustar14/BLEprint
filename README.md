## üöÄ Quick Start

This website is designed to be deployed on GitHub Pages. Follow these steps:

### 1. Create GitHub Repository

```bash
# Create a new repository on GitHub named "bleprint" or "your-username.github.io"
# Clone it to your local machine
git clone https://github.com/YOUR-USERNAME/bleprint.git
cd bleprint
```

### 2. Add Website Files

Copy these files to your repository:
- `index.html` - Main website page
- `style.css` - Styling and animations
- `script.js` - Interactive features
- `profileK.png` - Founder 1's professional photo
- `profileZ.png` - Founder 2's professional photo
- `README.md` - This file

### 3. Add Your Profile Photos

Replace the placeholder profile photos with your actual professional photos:

1. Rename your photos to `profileK.png` and `profileZ.png`
2. Place them in the root directory of the repository
3. Recommended size: 400x400px or larger (square aspect ratio works best)
4. Format: PNG or JPG

### 4. Deploy to GitHub Pages

```bash
# Add all files
git add .

# Commit changes
git commit -m "Initial BLEprint YC application website"

# Push to GitHub
git push origin main
```

### 5. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** ‚Üí **Pages**
3. Under "Source", select **main** branch
4. Click **Save**
5. Your site will be live at `https://YOUR-USERNAME.github.io/bleprint/` (or `https://YOUR-USERNAME.github.io/` if using username.github.io repo)

## üìù Customization

### Update Founder Names

Edit `index.html` and replace "Founder" with actual names in the team section:

```html
<h3>Your Name Here</h3>
<p class="team-role">Co-Founder & Technical Lead</p>
```

### Update Contact Email

In the contact section, update the email link:

```html
<a href="mailto:your-email@bleprint.ai" class="contact-btn primary">Get in Touch</a>
```

### Customize Colors

Edit `style.css` to change the color scheme:

```css
:root {
    --primary: #FF6B35;    /* Orange accent */
    --secondary: #004E89;  /* Dark blue */
    --accent: #F7B801;     /* Yellow highlight */
    --dark: #1A1A2E;       /* Dark background */
}
```
