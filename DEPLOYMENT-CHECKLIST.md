## Common Issues & Solutions

### Images Not Loading
**Problem:** Profile photos show placeholder
**Solution:** 
1. Check filenames are exactly `profileK.png` and `profileZ.png`
2. Ensure files are in root directory (not in a subfolder)
3. Clear browser cache and refresh

### Site Not Found
**Problem:** 404 error when visiting GitHub Pages URL
**Solution:**
1. Wait 3-5 minutes after enabling Pages
2. Check repository is Public
3. Verify `index.html` is in root directory
4. Try clearing browser cache

### Broken Links
**Problem:** Navigation doesn't work
**Solution:**
1. Ensure all section IDs match href values
2. Check JavaScript loaded (`script.js`)
3. Test in different browser

### Mobile Issues
**Problem:** Layout broken on phone
**Solution:**
1. Test in Chrome DevTools mobile view
2. Check viewport meta tag exists
3. Verify responsive CSS media queries
